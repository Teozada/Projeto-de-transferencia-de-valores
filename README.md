# DESCRIÇÃO

Este projeto é um teste Back-End para a Digital Republic

### Linguagem

- Java 11
- SpringBoot

# Clone

``` bash
# Clonar repositório

$ git clone https://github.com/Teozada/ProvaDigitalRepublic.git

# Entrar na pasta

$ cd ./ProvaDigitalRepublic

```

## Rodar a aplicação com DockerCompose

```bash

# Buildar a imagem

$ docker-compose up --build

```
